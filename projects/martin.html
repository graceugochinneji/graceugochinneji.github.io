Siamese CNN Model for
Melanoma Skin-Cancer Recognition

Project Overview

Jinzhou Xie develops a two-branch Siamese convolutional network that pairs identical Mod-VGG16 backbones. By measuring L1 distance between the twin embeddings and training on 10 605 dermoscopic images, the system reaches 91 % accuracy in distinguishing benign and malignant melanoma lesions. :contentReference[oaicite:0]{index=0}

Key Contributions

  • Designed a Mod-VGG16 sub-network (5 conv blocks + Flatten) to down-sample 105 × 105 inputs efficiently. :contentReference[oaicite:1]{index=1}
  • Built a full Siamese architecture with contrastive loss to learn lesion similarity.
  • Applied resizing, flip, HSV jitter & mosaic augmentation for robustness.
  • Trained with 90 % of data, validated on 10 %  → final accuracy 91 %, loss 25 %. :contentReference[oaicite:2]{index=2}
  • Delivered a PyQt5 GUI allowing clinicians to drop two images and receive a similarity score in <1 s.

Results Snapshot

  • Accuracy : 91.0 %
  • Validation Loss : 0.253
  • Confusion-Matrix F1 : 0.90 (benign), 0.96 (malignant) :contentReference[oaicite:3]{index=3}
  • GUI Test : >95 % passes across four clinical scenarios.

Limitations

  • Score threshold requires calibration per clinical workflow.
  • Occasional false positives when lesions are heavily occluded by body hair.
  • Dataset from a single portal → needs multi-centre validation.

Future Work

  • Expand dataset & retrain with self-supervised pre-text tasks.
  • Add lesion-segmentation pre-processing for higher precision.
  • Deploy mobile version with TensorFlow Lite quantisation.
