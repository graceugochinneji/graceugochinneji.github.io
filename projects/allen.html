Project Overview

Yiqi Leng built a 16-layer VGG-based CNN that learns to recognise all 47 classes in the EMNIST Balanced dataset (upper-, lower-case letters & digits). After 20 epochs of training with batch-normalisation, dropout 0.5 and Adam (LR = 1 × 10-4), the network reaches a remarkable 99.28 % accuracy on 18 800 held-out test images. :contentReference[oaicite:0]{index=0}

Key Contributions

  • Designed a deep VGG-16 pipeline (13 conv + 3 FC) with small 3 × 3 kernels and max-pooling.
  • Used EMNIST Balanced (112 800 train / 18 800 test) with on-the-fly normalisation.
  • Implemented dropout and L2 regularisation to avoid over-fitting; training + validation losses converge smoothly.
  • Delivered a Tkinter GUI where users can upload an image or scribble a letter for instant prediction.

Results Snapshot

  • Test accuracy : 99.28 %
  • Precision / Recall / F1 : 0.993 / 0.993 / 0.993
  • ROC-AUC : ≈ 1.00 across 47 classes
  • Confusion-matrix errors : < 0.7 % mis-classifications overall

Limitations

  • Model confidence drops on extremely faint strokes or oversized blobs.
  • No explicit bias-check across handwriting styles; domain shift to cursive remains untested.

Future Work

  • Port to MobileNetV3-Small for on-device OCR.
  • Add adaptive thresholding to boost performance on low-contrast ink.
  • Extend to multilingual alphabets (Greek, Cyrillic) with transfer-learning.
