Protein Secondary Structure Prediction Using Triple Fusion Explainable Model

What is this project about?

This project by Cecilia Chu presents a novel deep learning approach for predicting protein secondary structure. The model, termed the Triple Fusion Explainable Model, integrates a Convolutional Structural Predictor (CSP), a Self-Attention CSP, and a Memory-Based Network. The model provides interpretable predictions via SHAP visualizations, enabling better understanding of structural patterns from protein sequences.

How was it implemented?

The proposed model combines three components:

  • CSP: Extracts spatial features through convolutional layers.
  • Attention-CSP: Enhances CSP with self-attention mechanisms.
  • Memory-Based Network: Uses recurrent memory cells to retain long-term dependencies in protein sequences.

Training included real-time data augmentation, dropout regularization, and early stopping. The datasets were preprocessed with normalization and sequence filtering. Performance was evaluated using accuracy, AUC, and SHAP interpretability scores.

What were the key results?

The model achieved over 95% accuracy and AUC across both datasets. SHAP bar, dot, and waterfall plots revealed which features most influenced the structure classification.

Application & Future Work

A user-friendly GUI was developed to allow researchers to input protein sequences and receive predicted structures with SHAP-based insights. Future enhancements may include multi-modal sequence alignment, 3D structural prediction, and incorporation of protein function annotations.
