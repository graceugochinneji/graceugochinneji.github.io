Protein Secondary Structure Prediction Using Triple Fusion Explainable Model

What is this project about?

This project by Cecilia Chu presents a novel deep learning approach for predicting protein secondary structure. The model, termed the Triple Fusion Explainable Model, integrates a Convolutional Structural Predictor (CSP), a Self-Attention Mechanism, and a Memory-Based Network. The model provides interpretable predictions via SHAP visualizations, enabling better understanding of structural patterns from protein sequences.

How was it implemented?

The proposed model combines three components:

  • CSP: Extracts spatial features through convolutional layers.
  • Attention-mechanism: Enhances CSP with self-attention mechanisms.
  • Memory-Based Network: Uses recurrent memory cells to retain long-term dependencies in protein sequences.

Training included real-time data augmentation, dropout regularization, and early stopping. The datasets were preprocessed with normalization and sequence filtering. Performance was evaluated using accuracy, AUC, and SHAP interpretability scores.

What were the key results?

The model achieved over 95% accuracy and AUC across both datasets. SHAP bar, dot, and waterfall plots revealed which features most influenced the structure classification.

Conclusion

This study presents a robust and interpretable framework for protein secondary structure prediction using deep learning. The integration of CNNs, attention, and memory-based components allowed for high classification accuracy and generalization across two datasets. The explainability provided by SHAP plots supports trust and transparency in model decisions, making the model suitable for biomedical and research applications.

Limitations

  • Relies on secondary structure labels; does not extend to tertiary structure or protein folding prediction.
  • Interpretability is limited to SHAP approximations and lacks biochemical-level insights.
  • Model training is computationally expensive due to triple-branch architecture.

Future Work

  • Extend the framework to predict 3D protein structures and binding affinities.
  • Incorporate multimodal biological data (e.g., amino acid properties, gene expression).
  • Optimize architecture for deployment on low-power devices or web applications.
  • Enhance interpretability through attention heatmaps and domain-specific visualization tools.
