Protein Secondary Structure Prediction Based on Triple Fusion Explainable Model

Project Overview

This project, developed by Cecilia Chu, implements a novel Triple Fusion Explainable Deep Learning Model integrating a Convolutional Structural Predictor (CSP), Self-Attention Convolutional Structural Predictor (Attention-CSP), and Memory-based Structure Network. The model predicts protein secondary structures from sequence data and uses SHAP for result explainability. It was trained and validated using two key datasets.

Model Features

  • Combines CNNs, Attention, and Memory-based RNNs
  • Utilizes SHAP for interpretability (bar, force, dot, and waterfall plots)
  • Achieved over 95% accuracy and AUC on both datasets
  • Implements early stopping and dropout for regularization

Deployment

A GUI has been developed allowing users to input or upload protein sequences and obtain structure classification results along with visual performance metrics and SHAP interpretations.
