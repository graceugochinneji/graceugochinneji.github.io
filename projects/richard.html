Plant-Leaf Diseases Recognition Using Attention-Enhanced ResNet

Project Overview

Zichao Jin (Richard) presents a deep-learning pipeline that combines ResNet backbones with Channel & Spatial attention blocks for accurate recognition of 38 diseases across 14 crop species. The model is trained and evaluated on the New Plant Diseases dataset— an augmented edition of PlantVillage—achieving industry-leading performance while remaining compact enough for web deployment via Flask.

Key Contributions

  • Designed a ResNet + CBAM architecture that fuses channel- and spatial-attention for fine-grained lesion focus.
  • Employed extensive rotation, flip, crop and color jitter to counter class imbalance and improve generalisation.
  • Introduced a custom learning-rate scheduler and loss-function switch (MSE → Cross-Entropy) to eliminate over-fitting.
  • Integrated Grad-CAM visualisations and shipped a Flask GUI for farmer-friendly, drag-and-drop diagnosis.

Results Snapshot

  • Accuracy : 99.29 %
  • Precision / Recall / F1 : 0.99 / 0.99 / 0.99
  • ROC-AUC : 1.00  Average Precision : 0.99

The model ranks 2nd among recent state-of-the-art plant-disease classifiers and surpasses EfficientNet, DenseNet and previous attention-ResNet baselines on the same dataset.

Limitations

  • Dataset is lab-captured; performance on in-field, low-light images remains to be verified.
  • 38-class softmax occasionally confuses visually similar rusts and blights (<1 % error).
  • Training still requires a mid-range GPU (~11 M parameters).

Future Work

  • Collect real-world smartphone photos and retrain with domain-adaptation.
  • Prune & quantise the network for Raspberry Pi / Jetson-Nano edge deployment.
  • Expand the Flask app into a RESTful API for integration with farm-management platforms.
