Ensemble Learning for Accurate
Alzheimer’s MRI Classification

Project Overview

Leslie (Liu Yue) built an ensemble CNN that merges ResNet-50, AlexNet and MobileNet to diagnose Alzheimer’s disease from axial MRI slices. Combining the complementary strengths of the three backbones lifts performance and guards against the occasional failure of any single model, reaching a top accuracy of 98.2 % on a four-class task (non-, very-mild, mild and moderate dementia).

MRI Dataset GitHub Repo Download Poster

Key Contributions

  • Fine-tuned ResNet-50, AlexNet and MobileNet on 33 984 MRI slices (train : val : test ≈ 60 : 20 : 20).
  • Used Bagging to balance classes and combined model outputs with majority voting.
  • Achieved 98.2 % accuracy  (Macro F1 = 0.982)—a 3-4 pp lift over the best individual network.
  • Packaged a Flask GUI that loads DICOM/JPEG scans and displays the ensemble’s prediction & probabilities.

Results Snapshot

  • Accuracy : 98.2 %
  • Macro Precision / Recall / F1 : 0.98 / 0.98 / 0.98
  • AUC : 0.996  Inference time : 42 ms per slice (GPU)

Limitations

  • MRI data from a single scanner → needs cross-centre validation for domain robustness.
  • False positives on severe motion-blur slices.
  • Ensemble (~83 M params) still heavy for edge deployment.

Future Work

  • Distil the ensemble into a lightweight mobile model.
  • Add attention to highlight hippocampal atrophy regions.
  • Incorporate clinical metadata for multi-modal prognosis.
