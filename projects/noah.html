DeepSeg-ASPP + Transformer
Semantic Segmentation Framework

Project Overview

Xiang Feng proposes DeepSeg-ASPP + Transformer, a high-performance framework for real-time semantic segmentation in autonomous-driving scenarios. The network fuses a depthwise-separable ASPP module, a Transformer encoder for long-range dependencies, edge-detection and context-enhancement sub-blocks, all atop a ResNet-50 backbone. Trained on Cityscapes, the model balances accuracy with efficiency to meet real-time demands in complex urban scenes.

Key Contributions

  • DWC-ASPP: depth-wise separable Atrous Spatial Pyramid Pooling for multi-scale context with 40 % fewer FLOPs.
  • Transformer encoder: 4-head attention layer captures global dependencies beyond CNN receptive fields.
  • Edge & Context modules: SE-augmented edge block and multi-scale context enhancement improve boundary fidelity.
  • 11-class Cityscapes results: mIoU = 0.77, PA = 0.93, FPS ≈ 63 on RTX 3090 Ti (384 × 384 input).
  • Flask-based GUI enables drag-and-drop image / video / webcam segmentation in real time.

Results Snapshot

  • Accuracy : 93.6 %
  • mIoU : 0.77  Dice : 0.85  Kappa : 0.90
  • Precision / Recall : 0.95 / 0.97
  • Throughput : 63 FPS @ 384 × 384 (RTX 3090 Ti)

Limitations

  • Evaluated only on Cityscapes—generalisation to other domains (e.g., KITTI, BDD100K) unverified.
  • Transformer block adds ~15 % VRAM overhead at higher resolutions.
  • Edge cases (heavy rain, night driving) still cause occasional mis-segmentations.

Future Work

  • Domain-adaptation training for cross-city and adverse-weather robustness.
  • Model pruning & quantisation for embedded deployment (Jetson Orin / Xavier).
  • Incorporate self-supervised pre-training to reduce labelled-data requirements.
