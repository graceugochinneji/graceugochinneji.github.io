Design & Implementation of a Web-based

Project Overview

Shuai Gan designed and built a multilingual e-commerce site that pairs a Vue.js single-page front end with a Django + MySQL back end. Key features include secure payment handling, language switching (English/Chinese), role-based dashboards, and just-in-time delivery scheduling.:contentReference[oaicite:0]{index=0}
