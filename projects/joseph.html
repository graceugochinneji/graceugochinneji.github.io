Deep-Learning Application for the
Recognition of Breast Cancer (IDC)

Project Overview

Joseph Zhang built an AlexNet-based CNN to detect Invasive Ductal Carcinoma (IDC) from histopathology patches. After balancing the publicly-available Breast Histopathology dataset and applying flips & resizing, the network achieved an 84.7 % accuracy on validation images.:contentReference[oaicite:0]{index=0}

Key Contributions

  • Balanced the dataset to ~20 k positive / 20 k negative patches before training.:contentReference[oaicite:1]{index=1}
  • Customised AlexNet for binary classification with 224 × 224 inputs.
  • Implemented Tkinter GUI for drag-and-drop IDC prediction in clinics.

Results Snapshot

  • Accuracy : 84.7 %
  • Precision : 95.0 %
  • Recall : 39.0 %
  • F1-score : 55.3 %

Limitations

  • Recall is low; model favours negative predictions.
  • Trained on 40× patches only—needs multi-magnification validation.

Future Work

  • Add ROC/AUC evaluation & larger batch for steadier loss curves.
  • Explore deeper backbones (ResNet, EfficientNet) to raise recall.
