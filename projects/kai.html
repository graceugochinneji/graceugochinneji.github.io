Human Fall Detection System using Deep Learning

Project Overview

This project tackles the growing need for rapid, reliable fall detection among older adults. Zhang Zhirui (Kai) combines YOLO v7 object detection and OpenPose key-point tracking with a dual-channel CNN ensemble to recognise falls in real time and alert caregivers.

Key Contributions

  • Pre-processing pipeline with YOLO v7 + OpenPose to isolate bodies and 18 skeletal key-points.
  • Two-channel sliding-window model: dynamic features (centre-of-mass & limb velocities) + static contour features.
  • Ensemble of ResNet-with-Channel & Spatial Attention and a custom Kai_CNN branch.
  • MLP + Random Forest classifiers fuse feature streams for robust fall / non-fall discrimination.
  • Grad-CAM visualisations and a Tkinter GUI deliver interpretable, operator-friendly feedback.

Results Snapshot

  • Ensemble Accuracy: 99 %
  • Precision / Recall / F1: 0.99 / 0.99 / 0.99
  • ROC-AUC & PR-AUC: 1.00 (perfect separation)
  • Outperforms baseline ResNet50, VGG16, EfficientNetB0 and MobileNet by ≥ 15 %.

The high sensitivity ensures virtually every real fall is caught, while a 96 %+ specificity keeps false alarms rare— ideal for in-home or clinical monitoring.

Conclusion

By fusing pose-based dynamics with appearance cues, the system achieves state-of-the-art fall detection on modest GPU hardware, laying the groundwork for affordable, scalable elder-care solutions.

Limitations

  • Dataset sourced from a single environment—domain shift to new camera angles or lighting untested.
  • Extreme occlusions or crowded scenes may reduce accuracy.
  • Current pipeline requires an RTX-class GPU for real-time processing.

Future Work

  • Add transformer-based pose models for better occlusion handling.
  • Deploy compressed (pruned / quantised) networks to edge devices (Jetson, Raspberry Pi + NPU).
  • Integrate temporal LSTM layers to further stabilise predictions over video streams.
