Explainable Deep Learning for Drug-Target Interaction Prediction Using Protein Structures

Student: Kerui Liu (Kyrie)  |  Supervisor: Dr. Grace Ugochi Nneji

Institution: Chengdu University of Technology - Oxford Brookes College

Abstract

This project presents a hybrid deep learning framework for predicting drug-target interactions (DTIs) using protein structures. The model integrates a Frequent Continuous Substructure (FCS) mining module, Transformer encoders, and CNNs. Trained on BIOSNAP, BindingDB, and DAVIS datasets, the model achieves high AUC, AUPRC, and F1 scores. Explainability is ensured via SHAP and attention maps. A GUI interface supports real-time predictions and visualizations, enhancing accessibility and interpretability for biomedical research.

Methodology

  • Datasets: BIOSNAP, BindingDB, and DAVIS
  • Preprocessing: BPE encoding, padding, masking
  • Model Components:
    • FCS Mining Module (tokenizes SMILES and protein sequences)
    • Transformer Embedding Module (captures sequence dependencies)
    • Interaction Prediction Module (2D CNN + MLP)
  • Training Details: Adam optimizer, BCE loss, dropout, early stopping
  • Evaluation Metrics: AUC, AUPRC, F1 Score, Loss

Results

The final model achieves the following on BIOSNAP:

  • AUC: 0.8853
  • AUPRC: 0.8881
  • F1 Score: 0.8287
  • Loss: 0.4683

Performance on DAVIS and BindingDB further confirms its robustness across datasets.

Explainability

SHAP plots (Force, Summary, Waterfall, Dependency, Bar) were generated for interpretation. Features with the highest impact were identified, and visualization tools assist in understanding individual predictions.

GUI Demo

An interactive Streamlit web interface was built, supporting input of SMILES and protein sequences for real-time predictions, including model visualization, confusion matrix, SHAP results, and probability outcomes.

Conclusion

This project introduced a hybrid deep-learning model integrating FCS, Transformer, and CNN for interpretable and effective drug-target interaction prediction. Tested across multiple datasets, it demonstrated high performance, robust generalization, and strong explainability. The deployment of a GUI makes the system user-friendly and practical for drug discovery research.

Limitation

The model uses fixed-length padding, which may lose contextual information. Class imbalance persists, especially with rare interactions, and though mitigated with loss weighting, remains an issue. Additionally, the current convolutional architecture may miss finer-grained structural features.

Future Work

Future efforts will explore integrating GNNs, 3D protein structure encodings, and transfer learning to enhance generalization. Further optimization of model efficiency and dynamic adaptation to new datasets are also planned.
