Separable-CNN Inception-ResNet for Enhanced Breast-Cancer Classification

Project Overview

ZiXuan Chen proposes a hybrid deep-learning pipeline that merges Separable Convolutions, Inception modules, ResNet residual blocks, and a lightweight channel-&-spatial attention (CBAM) layer. Trained on the BreaKHis 400× histopathology set, the ensemble targets early-stage breast-cancer detection while keeping parameter count manageable for clinical deployment.

Key Contributions

  • Introduced Separable-CNN blocks inside Inception paths to cut FLOPs without losing spatial detail.
  • Stacked residual (ResNet-style) skip connections to stabilise deep training.
  • Inserted CBAM attention after each macro-block for fine-grained focus on malignant tissue regions.
  • Employed heavy augmentation plus class-rebalancing to tame BreaKHis’ benign/malignant skew.
  • Released a Flask-based demo with Grad-CAM overlays for pathologist interpretability.

Results Snapshot

  • Accuracy : 96.4 %
  • Sensitivity : 0.968  Specificity : 0.977
  • Precision : 0.974  F1-Score : 0.971
  • ROC-AUC : 0.99 (macro)

The proposed ensemble surpasses vanilla VGG16, ResNet-50, Inception-v3 and several recent DenseNet/EfficientNet baselines reported on the same dataset.

Limitations

  • Relies on a single-centre dataset—domain shift to other labs remains untested.
  • High-res training (400×) demands a 3070 Ti-class GPU for < 8 hour convergence.
  • No bias-mitigation yet for under-represented histological subtypes.

Future Work

  • Cross-institution validation with multi-magnification slides.
  • Knowledge-distillation to a mobile-size model for point-of-care microscopes.
  • Self-supervised pre-training to shrink annotated-data requirements.
