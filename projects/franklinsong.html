Depthwise-Separable Inception Network
for Skin-Disease Classification

Project Overview

Franklin Song designs a lightweight Depthwise Separable Inception Network to diagnose seven common dermoscopic categories in the HAM10000 dataset. The architecture fuses Inception-v4 modules with depthwise separable convolutions, slashing FLOPs and parameters while retaining high accuracy (82 %) and faster inference—ideal for clinical and mobile scenarios.

Key Contributions

  • Merged depthwise separable convolutions with Inception-v4 Stem + A/B/C blocks for a 40 % parameter cut.
  • Balanced the imbalanced dataset via heavy augmentation (rotation ±40°, shifts 20 %, shear/zoom 20 %).
  • Achieved 82 % test accuracy  (F1 = 0.79, ROC-AUC ≈ 0.93)—up to +7 pp over vanilla Inception.
  • Provided Grad-CAM heat-maps for clinical interpretability and a Flask image-upload GUI.

Limitations

  • Training data from a single centre; cross-institution generalisation still untested.
  • Accuracy drops on low-light smartphone photos.
  • Requires mid-range GPU (< 8 h at 224 × 224) for full-resolution training.

Future Work

  • Cross-institution validation & multi-magnification training.
  • Pruning/quantisation for Raspberry Pi / Jetson edge deployment.
  • Self-supervised pre-training to cut labelled-data demands.
