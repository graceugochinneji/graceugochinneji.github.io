Automatic License-Plate Detection
with Improved YOLOv5 + OCR

Project Overview

Xi Zhu (student # 201918020310) built a two-stage pipeline that first localises license plates with an improved YOLOv5 detector — adding ASPP, adaptive anchors and mosaic augmentation — then extracts the plate patch and runs Tesseract OCR for character recognition. The system handles varied lighting, skew and motion blur, reaching 88 % end-to-end accuracy on four combined public datasets.

Key Contributions

  • Enhanced YOLOv5s with DWC-ASPP + mosaic augmentation for small-plate detection.
  • Adaptive letter-box resizing that adds minimal padding, boosting FPS ~10 %.
  • Curated & merged 4 datasets / 3 282 images; performed custom tiling & label harmonisation.
  • Delivered 79 % mAP@0.5  |  Precision 0.80 · Recall 0.78 · F1 0.79 for detection; overall system accuracy 88 %.
  • Packaged a Flask web demo for real-time plate reading on uploaded images/videos.

Results Snapshot

  • Detection mAP@0.5 : 0.79
  • End-to-end license-plate accuracy : 88 %
  • Inference speed : 38 ms per frame (RTX 2060, 640×640)
  • False-alarm / miss rate : 5 % / 7 %

Limitations

  • Performance drops on night-time IR footage & extremely oblique angles.
  • OCR errors for plates under heavy motion blur or mud splatter.
  • Current pipeline (~42 MB) still large for microcontroller deployment.

Future Work

  • Add transformer-based detector (YOLOv8-N, RT-DETR) for long-range context.
  • Integrate liveness / spoof-prevention & domain-adaptation for night scenes.
  • Distil model + quantise to INT8 for edge devices (Jetson Nano).
