Project Overview

Olivia's project introduces Attention-ProteinMeNet, a hybrid deep learning architecture for predicting protein secondary structure. The model combines a deep CNN with a spatial-channel attention mechanism and residual connections to capture both sequential dependencies and contextual structure from protein sequences.

Key Contributions

  • Proposed a CNN-Attention hybrid model called Attention-ProteinMeNet.
  • Evaluated model performance on the CB513 and RCSB-PDB datasets.
  • Integrated spatial and channel-wise attention to enhance structure localization.
  • Achieved high classification accuracy and generalization on unseen sequences.
  • Implemented a GUI tool for sequence input and visual structure prediction output.

Conclusion

Attention-ProteinMeNet improved structure prediction accuracy through attention refinement. The model achieved over 85% accuracy on CB513 and effectively generalized to sequences from RCSB-PDB. The combination of convolutional depth, attention, and skip connections yielded robust performance for biomedical applications.

Limitation

  • Model complexity increases inference time for longer protein sequences.
  • Dependent on curated alignment features, which may limit real-time use.
  • Struggles with rare or disordered protein configurations.

Future Work

  • Integrate transformer-based encoders for global dependency modeling.
  • Expand training with noisy sequences to improve robustness.
  • Deploy the model for web-based usage in structural biology labs.
