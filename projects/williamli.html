Harnessing Inception-Depthwise-Attention CNN + Residual Learning
for Improved Skin-Cancer Classification

Project Overview

Xinglin Li (William) proposes a composite network that fuses depthwise separable convolutions, Inception multi-scale paths, ResNet skip connections and a channel–spatial attention block to boost diagnostic accuracy on dermoscopic images. :contentReference[oaicite:0]{index=0}

HAM10000 Dataset GitHub Repo Download Poster

Key Contributions

  • Introduced a Depthwise-Inception-ResNet-Attention block that reduces FLOPs while preserving rich multi-scale features. :contentReference[oaicite:1]{index=1}
  • Applied aggressive data augmentation and class-rebalancing to address HAM10000’s category imbalance. :contentReference[oaicite:2]{index=2}
  • Achieved 98.64 % test accuracy, F1 = 0.986 and ROC-AUC ≈ 0.99, outperforming VGG, EfficientNet and earlier attention-CNN baselines. :contentReference[oaicite:3]{index=3}
  • Released a Grad-CAM–enabled Flask GUI for clinician-friendly, explainable predictions.

Limitations

  • Model trained on a single-centre dataset; cross-institution generalisation still unverified. :contentReference[oaicite:4]{index=4}
  • Performance drops on low-light, in-field smartphone images; future work will add domain-adaptation pipelines.
  • Requires a mid-range GPU for < 8 h convergence at 224 × 224 resolution.

Future Work

  • Introduce self-supervised pre-training and GAN-based augmentation to cut annotation demand.
  • Prune & quantise network for Raspberry Pi / Jetson-Nano edge deployment.
  • Validate on ISIC-2019 and real-world clinic photos to benchmark domain shift.
