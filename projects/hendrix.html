Deep-Learning Approach for
Skin-Cancer Classification

Project Overview

Hanmin Huang evaluates four ResNet backbones (18 / 34 / 50 / 101) on the dermoscopic HAM10000 dataset. After oversampling minority classes with SMOTE and augmenting images, ResNet-18 tops the leaderboard with 96.47 % accuracy, proving a light-weight network can match—or beat—much deeper models. :contentReference[oaicite:0]{index=0}

Key Contributions

  • Balanced the seven-class dataset (10 015 → 46 935 images) via SMOTE oversampling. :contentReference[oaicite:1]{index=1}
  • Compared ResNet-18/34/50/101 under identical hyper-parameters for a fair bake-off.
  • Achieved the best metrics with ResNet-18: Accuracy 96.47 %, F1 96.43 %, Precision 96.47 %. :contentReference[oaicite:2]{index=2}
  • Delivered a Tkinter GUI for drag-and-drop lesion diagnosis in < 1 s per image.

Results Snapshot

  • Accuracy : 96.47 %
  • F1-score : 96.43 %
  • Precision / Recall : 96.47 % / 96.44 %
  • Best confusion-matrix corrects : 9 006 / 9 387 samples (ResNet-18). :contentReference[oaicite:3]{index=3}

Limitations

  • Recall still trails precision—model skews slightly toward benign predictions.
  • Training limited to 40× magnification; multi-scale validation pending.
  • Edge-case images with heavy hair occlusion or low light occasionally misclassify.

Future Work

  • Integrate attention modules to improve malignant recall.
  • Test transfer-learning on ISIC 2019 for multi-centre robustness.
  • Experiment with EfficientNet-B0 for a lighter mobile deployment.
