Multi-Modal Deep Learning for Alzheimer’s Disease Diagnosis from MRI aand Clinical Data

Student: Wang Yifei (Peter)  |  Supervisor: Dr. Grace Ugochi Nneji

Institution: Chengdu University of Technology - Oxford Brookes College

Project Overview

This project proposes a novel multimodal deep learning framework that combines MRI neuroimaging with clinical metadata for early Alzheimer's disease (AD) diagnosis. By integrating convolutional neural networks with squeeze-and-excitation (SE) attention mechanisms, the model prioritizes key pathological markers like hippocampal atrophy. It achieves 94.8% accuracy and operates via a Flask-based GUI that offers sub-3-second predictions.

Key Contributions

  • Developed a CNN-SE architecture with attention mechanisms for enhanced biomarker localization.
  • Integrated clinical data (MMSE, APOE-ε4) for multimodal prediction.
  • Achieved 94.8% accuracy, 95.2% sensitivity, and 93.7% specificity.
  • Implemented SHAP and Grad-CAM visualizations for interpretability.
  • Designed a Flask-based GUI for clinician-friendly interaction.

Conclusion

The multimodal deep learning framework successfully bridges imaging and clinical data to enhance Alzheimer’s diagnosis. Attention mechanisms improved focus on critical brain regions while SHAP plots ensured interpretability. Clinical deployment is enabled through a lightweight and user-friendly web interface.

Limitation

  • High GPU computation time (48 hours for training).
  • Dataset diversity is limited—mostly North American and European cohorts.
  • Requires further validation via clinical trials.

Future Work

  • Train on more diverse populations using federated learning.
  • Incorporate mobile-friendly CNN architectures like MobileNet.
  • Develop natural language reporting features for clinician insights.
