Inception + Depthwise-Separable CNN
for Retinal-Disease Classification

Project Overview

Yifan (Eve) Ke proposes a lightweight network that merges Inception-V3 modules with depthwise-separable convolutions to diagnose four retinal conditions (cataract, diabetic retinopathy, glaucoma, and normal) from OCT images. The hybrid design captures multi-scale features while trimming FLOPs and parameters, delivering fast, accurate screening suitable for clinical decision support.

Key Contributions

  • Designed a five-stage Inception A→E backbone enhanced with depthwise-separable convolutions for 40 % fewer parameters.
  • Used aggressive augmentation (random crop, flip, ±20° rotation) and class balancing to curb data imbalance.
  • Achieved 91.9 % validation accuracy with F1, Precision, Recall ≈ 0.91 on the four-class OCT set.
  • Bench-marked against ResNet, VGG-16, MobileNetV2 and outperformed all except a tuned MobileNet-v2 while using less compute.
  • Implemented a Flask GUI with LIME explanations for clinician-friendly, explainable predictions.

Results Snapshot

  • Accuracy : 91.9 %
  • Average Precision / Recall / F1 : 0.91 / 0.91 / 0.91
  • Validation Loss : 0.057
  • Confusion-matrix analysis shows balanced performance across all four classes.

Limitations

  • Single-centre OCT dataset — domain shift to other scanners still untested.
  • Model accuracy dips on severely noise-corrupted or low-light scans.
  • No hyper-parameter sweep for optimiser / schedule; potential gains remain.

Future Work

  • Cross-institution validation with multi-magnification OCT images.
  • Knowledge-distillation to a micro-model for on-device (mobile) diagnostics.
  • Integrate self-supervised pre-training to cut labelled-data demand.
