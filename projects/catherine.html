Dog Breed Classification
using Convolutional Neural Networks

Project Overview

Fanlin Zhou compares three transfer-learning backbones—VGG-16, ResNet-50 and Xception—on 10 k images spanning 120 dog breeds. After extensive augmentation and hyper-tuning, Xception edges ahead with 91 % top-1 accuracy, while a fine-tuned VGG-16 reaches 99 % accuracy on the training set for rapid identification in shelters, vet clinics and mobile apps.:contentReference[oaicite:0]{index=0}

Key Contributions

  • Evaluated VGG-16, ResNet-50 & Xception with transfer-learning on 120 breeds.
  • Applied rotation, shear, zoom & color-jitter to balance long-tail classes.
  • Fine-tuned VGG-16 to 99 % train / 92 % val accuracy; Xception topped 91 % top-1 on test.:contentReference[oaicite:1]{index=1}
  • Released a Flask drag-and-drop web demo with Grad-CAM visual explanations.

Results Snapshot

  • VGG-16 : 99 % train · 92 % val
  • ResNet-50 : 95 % train · 81 % val
  • Xception : 92 % train · 72 % val
  • Macro F1 (best) : 0.89

Limitations

  • Performance dips on low-light or multi-dog scenes.
  • 83 M-param ensemble too heavy for edge micro-controllers.

Future Work

  • Distil ensemble to a lightweight mobile model.
  • Add attention for pose cues and mixed-breed detection.
  • Gather in-the-wild photos for cross-domain robustness.
